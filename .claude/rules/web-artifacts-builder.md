# Web Artifacts Builder Rules

Bu kurallar sohbet içinde veya dashboard üzerinde zengin bileşenler üretirken uygulanır.

## 1. Bileşen Kuralları
- **Etkileşim:** Statik HTML yerine kullanıcının tıklayabileceği, filtreleyebileceği dinamik yapılar kur.
- **Görsel Kalite:** `frontend-design` kurallarına sadık kal.
- **Kendi Kendine Yetme:** Bileşenler dışarıdan minimum bağımlılık alacak şekilde (Self-contained) tasarlanmalıdır.
