❯ yeni bir mail ile claude ye nasıl bağlanabilirim                                                  

⏺ Claude'a yeni bir mail ile bağlanmak için şu adımları izleyebilirsiniz:
          
  Claude.ai (Web)

  1. claude.ai adresine gidin
  2. Sağ üst köşeden çıkış yapın (Sign out)
  3. "Sign up" ile yeni mail adresinizi kullanarak kayıt olun
  4. Ya da "Sign in" → "Continue with Google/email" ile farklı hesap seçin

  Claude Code (CLI)

  claude logout
  claude login

  Komutu çalıştırdığınızda tarayıcı açılır, oradan yeni mail ile giriş yapabilirsiniz.

  API Kullanımı

  Yeni mail ile console.anthropic.com adresinde hesap oluşturup yeni bir API key üretebilirsiniz.
  Ardından:

  # .env veya shell config dosyasına
  ANTHROPIC_API_KEY=sk-ant-...yeni-key...
